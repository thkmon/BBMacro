readme.txt for BBMacro
----------

■ 실행에 필요한 파일(필수파일)
실행에 필요한 파일은 BBMacro.bat, BBMacro.jar, command.txt 파일입니다.
필요 JDK 버전은 1.6 입니다.

■ 실행에 필요하지 않은 파일
지금 읽고 있는 readme.txt 파일은 사용설명서입니다.
나머지 bbmacro, img 폴더 등은 테스트를 위한 데이터이므로 나중에 필요하지 않을 경우 삭제해도 됩니다.
log 폴더는 실행로그가 쌓이는 폴더입니다. 용량문제 등으로 로그를 쌓고 싶지 않으면 폴더를 삭제하면 됩니다.

■ 기본적인 테스트 방법
동봉되어 있는 bbmacro 폴더를 C 드라이브에 붙여넣기 합니다. (C:\bbmacro 경로가 되도록 붙여넣기하면 됨)
이후 BBMacro.bat 파일을 실행합니다.

컴퓨터 환경에 따라 정상실행되지 않을 수 있습니다.
예를 들어 테스트 로직은 한글 2010을 이용한 매크로이므로, 한글 2010 미설치 PC의 경우 정상작동하지 않습니다.

■ 매크로 작성 방법
command.txt 파일 내용을 적절하게 수정해주세요.

■ 매크로 종료 방법
BBMacro.bat 파일을 실행해서 나타난 커맨드 창(cmd)을 종료하면 됩니다.

■ 문제해결
실행이 안될 경우 bat 파일 내의 jdk1.6 경로를 올바른 값으로 수정해주세요.
